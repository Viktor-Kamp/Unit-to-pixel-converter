"Unit to pixel converter" is a Blender Add-On (compatible since Blender 4.2 until latest). I created it, to be able to quickly calculate the needed render resolution from different units (like inch, centimeter or millimeter) inside Blender while taking PPI/DPI (pixel density) into account. This is especially useful for print production.

You can interact with the Add-On in: "Properties > Output > Unit to pixel converter"
